

What type of resources can you host with Express?
    Web applications, frameworks, APIs, Performance features.
What are the benefits of using a View Engine overusing Vanilla technologies?
    The application generator tool, express-generator, quickly creates an application skeleton rather than the slower method of doing it manually.
Why does the Express Generator split up the functionality into so many files?
    Splitting components apart makes it so that there can be consistency across an entire site, and the coding organized and easy to follow.
